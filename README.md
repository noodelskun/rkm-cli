##  一个简易的页面脚手架工具

新建项目
```
rkm create <name>
```

添加组件
```
rkm add <name> -c
```

添加页面
```
rkm add <name> -p
```

启动调试
```
rkm dev
```

打包构建
```
rkm build
```

### 自定义配置

自定义配置可在根目录下创建km.config.js，配置项参考webpack
